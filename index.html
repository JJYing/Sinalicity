Sinalicity

Minimalized Sina Weibo CSS Style

Download .zip Download .tar.gz View on GitHub

V5 版新浪微博推出之后和往常一样我自己动手用 Stylebot 进行了深度自定义，去掉很多我个人从来不点不看的区块并利用一些 CSS 的 transition 动画对很多元素进行了彻底的重新设计，整体的思路就是将最大的空间留给主 timeline，不常用但必须的操作区域尽量不显性化。本来想做得完整一点再发布，但人懒一直保持着半成品的样子，眼看着再不发出来新浪又要改版了，所以如果有朋友喜欢这种极简风格的就拿去吧~

部分特性：

  • 去除了广告、推广、热门话题等，只保留干净的阅读空间给主 timeline
  • 将个人信息、单条信息操作等区域进行了隐形化处理
  • 借鉴随享、Weico 等 App 将文字连接做成一个轻量化的按钮以净化主 timeline
  • 发帖区域进行了简化，当鼠标未悬浮时呈现极简和极小的状态
  • 顶条也做了部分删减和重新设计
  • 以白色为底灰色为主的页面风格设计

安装方法：

  1. 使用 Chrome 浏览器（ Win 和 Mac 均可 ）
  2. 安装 Stylebot 插件
  3. 前往此页面一键安装皮肤
